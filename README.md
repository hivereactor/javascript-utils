## @hivereactor/utils

Utility functions for your JavaScript project.

### Installation

```zsh
git clone https://github.com/hivereactor/javascript-utils.git
cd javascript-utils
npm link
```

In your project:

```zsh
npm link @hivereactor/utils
```
